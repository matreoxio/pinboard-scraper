# pinboard-scraper

###
Before compiling this project, make sure to download and deploy - https://github.com/matreoxio/pinboard-scraper-api

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Libraries used 
* Tailwind for styling
* Axios for making api requests
